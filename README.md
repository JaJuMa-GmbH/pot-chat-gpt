## Installation

Install via composer as any other Magento extension from Github:
```
composer require jajuma/pot-chat-gpt
```

## Using ChatGPT Power Toy For Magento

After installing this module:  
Go to   
**JaJuMa -> Power Toys -> Configuration**  
and enable & configure ChatGPT Toy.

To get a Chat GPT API Key, please register for OpenAI here:  
https://platform.openai.com/  
and create your API Key here:  
https://platform.openai.com/api-keys  

Please find valid API endpoints here:  
https://platform.openai.com/docs/guides/text-generation

Please find available models here:  
https://platform.openai.com/docs/models/overview

After configure and enabling, see the floating button at left/right edge of your screen 
in your Backend and Frontend (while logged in as Admin).  
When click on this button, the Power Toys Panel will open 
displaying the ChatGPT Toy.
A click on ChatGPT Toy will open the ChatGPT chat popup - Happy Chatting!

* [See Manual for more details](https://www.jajuma.de/media/wysiwyg/jajuma-develop/power-toys-magento/manuals/JaJuMa_Chat_GPT_Power_Toy_Manual_v001.pdf)
